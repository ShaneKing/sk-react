# Features
- Two-way Binding
  - component --- add listener --- ksModelId on model
  - ksModelId's value change --- fireEvent -- component auto refresh
  - component event --- auto change --- ksModelId's value
- Component props
  - model series: skSysModel(SPA just one), skModel(even page), ksModel(some scenario), ksModelId(auto two-way binding and monitor)
  - value process series: ksM2nConverter, ksN2mConverter
- props trans
  - auto trans to child(without ksXxx) and self(without ksXxx, skXxx if html tag)
